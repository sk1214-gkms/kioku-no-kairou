# アセット仕様書（画像・イラスト・音）— 記憶の回廊

> 実データ（全ステージJSON / cipher.json / アイテム）から抽出した「実際に必要なファイル」と、各部屋の描画内容・特徴を網羅。
> **ファイル名はコードと完全一致**させること（名前が一致すれば自動で読み込まれる想定）。

---

## 0. 基本仕様（最初に読む）

| 項目 | 指定 |
|---|---|
| プレイ領域 | **中央キャンバス 360×640（9:16・縦）**。背景はこの比率で作る |
| 解像度 | **@3x 推奨：背景 1080×1920**／ズーム詳細 800×800以上／アイコン 256×256 |
| 形式 | 背景＝PNG or JPG（不透明）／オブジェクト・アイコン・グリフ＝**PNG透過**（WebP可） |
| 座標 | interactables の `rect:[x,y,w,h]` は **360×640座標**。物は背景に描き込みでOK（rectはタップ領域）。**ズーム画像＝調べた時に出る拡大絵**は別物 |
| レイヤ構成 | 基本2層：**①背景（情景）＋②ズーム画像（クローズアップ）**。状態差分（明暗・通電など）は背景違い or オーバーレイ |
| 命名 | 本書の一覧どおり（例 `bg_stage_03.png` `clock_0345_zoom.png`） |

## 1. 全体アートディレクション

- **世界観**：1950〜60年代風の洋館——だが正体は「医療施設／蓮見の脳内に再構築された記憶の迷宮」。**ゴシックな洋館 × 無機質な医療**の二重性。
- **パレット**：暗い青紫〜セピア基調、**血の赤**がアクセント。**章が進むほど“現実(白い医療)”が滲み出す**（後半背景にノイズ・電極・モニタ等を混ぜる）。
- **テーマ演出**：要所で“現実が綻ぶ”表現（走査線ノイズ、白い病室的要素）。15号室以降は特に。
- **正常/虫食い**：テキストで分岐するので**別画像は不要**。演出したい場合のみ、共通の「ノイズ/欠損オーバーレイ」を1枚用意すると効果的（任意）。
- **統一感**：全背景は同一画風・同一光源傾向で。AI生成なら下記プロンプト核を共通化（§8）。

## 8. AIで作る際のコツ（方針：素材サイト＋AI生成）

- **統一プロンプト核**（例）：`dark vintage 1950s western mansion interior, muted teal-sepia palette, dim cinematic lighting, subtle medical/uncanny undertone, painterly, vertical 9:16`。これを全背景で共通化し**画風を固定**。
- **img2img**で素材サイトの写真/画像を同一画風に揃える（バラつき防止）。
- **オブジェクト/アイテムは透過背景**で生成（`on transparent background`）or 後で背景除去。
- **回廊文字グリフ**は「架空のルーン文字、18種、統一線幅、白黒透過」で一括生成。
- ⚠️ **商用ライセンスは素材側・AI側の双方で確認**（広告収益＝商用）。img2imgで素材を改変する行為が元ライセンスで許可されているかも要確認。
