> 現行の **深い部屋キャンペーン（作話システム）** に必要なアセットを、実データ [data/deep_rooms/](../data/deep_rooms/) から抽出。仕様は [仕様書_v3_作話システム.md](仕様書_v3_作話システム.md) を参照。

## 0. 基本仕様

| 項目 | 指定 |
|---|---|
| プレイ領域 | **中央キャンバス 360×640（9:16・縦）**。`DesignCanvas` 上で FittedBox スケール |
| 解像度 | @3x 推奨：背景 1080×1920／ズーム(subview/拡大) 800×800以上／アイコン 256×256 |
| 形式 | 背景＝PNG/JPG（不透明）／オブジェクト・アイコン＝PNG透過 |
| 座標 | `views.<dir>.objects[].rect:[x,y,w,h]` は 360×640座標 |
| **4方向構成** | 各部屋は **東西南北4視点**。原則 **1方向＝1背景**（最大 13×4＝52枚）。簡素な面は省略・流用可 |
| 状態差分 | toggle で見え方が変わる面のみ複数用意（§4の★） |
| HUD | 上部に脳死タイマー（残り時間）＋点灯済みトラウマ文字（黒文字体・§5） |

## 1. アートディレクション

- **世界観**：1950〜60年代風ゴシック洋館——だが正体は**瀕死の蓮見鏡介の脳内に再構築された記憶の迷宮**。洋館 × 無機質な医療の二重性。
- **パレット**：暗い青紫〜セピア、**血の赤**がアクセント。**章が進むほど“現実(白い医療)”が滲む**（後半に走査線ノイズ・電極・モニタ）。
- **後半の崩壊**：R10以降は赤い文字化け（VERLUST の予兆）を背景に混ぜると効果的。
- **記憶の正体は嘘**：情景はどこか作り物めいた違和感（不自然な反復、辻褄の合わない陰影）を忍ばせる。
- **統一感**：全背景同一画風・同一光源。AI生成は §8 のプロンプト核を共通化。

## 4. 部屋別 4方向アセット（R1〜R13）

> 表記：方向＝情景。`{ }`＝ズーム/subview画像。★＝状態差分（toggleで背景違いが要る面）。

### 第1章（R1〜R4）
- **R1 白い部屋**：北=洗面台`{洗面台の下：奥の小箱→鍵}`／東=ベッド(血の染み)／南=施錠扉／西=鉄格子の窓(深い霧)
- **R2 準備室**：北=薬棚`{引き出し奥：紙片左}`／東=作業台(紙片右)／南=通行証式の扉／西=ロッカー
- **R3 暗室**：西=配電盤(ブラックライト)★／北=現像壁`{要ライトON：0415}`★／東=止まった時計(03:45・罠)／南=数字錠の扉
- **R4 書斎**（分岐①）：北=氷室の肖像`{裏の金庫→ドアノブ}`／西=暗号パネル(獣の並び)／南=取付部の扉／東=本棚

### 第2章（R5〜R9）
- **R5 診察室**：北=カルテ(数字→文字)／東=血の付いた診察台／南=文字錠の扉／西=薬棚
- **R6 記録室**：北=容疑者表／東=証言A／西=証言B／南=数字錠の扉
- **R7 廊下**：北=白い足跡(本物)／東=血の足跡(decoy)／西=泥の足跡(decoy)／南=数字錠の扉
- **R8 鏡の間**（分岐②）：西=照明スイッチ★／北=穏やかな肖像／東=鏡`{要照明OFF：反転数字／返り血の自分}`★／南=数字錠の扉
- **R9 標本室**：北=標本棚`{奥：空シリンジ}`／東=薬液の瓶／西=隠し戸棚`{奥：金属製の筒}`／南=注射器を挿す扉

### 第3章（R10〜R13）
- **R10 監視室**：西=主電源★／北=モニターA`{要通電}`★／東=モニターB`{要通電：6271}`★／南=数字錠の扉
- **R11 手術室**：北=無影灯／東=カルテの束／西=トレイの器具(＋壁の引っ掻き傷)／南=数字錠の扉（部屋の“数”が暗号）
- **R12 証拠保管室**（分岐③）：北=証拠箱`{中：甥の手袋}`／東=事件ファイル／西=現場写真／南=数字錠の扉

## 5. GEDÄCHTNIS / VERLUST 演出（符号化）

- **トラウマ文字は画像ではなく“フォント”で描画**。`fontFamily:'Blackletter'`（**UnifrakturMaguntia** 等の黒文字体・OFL）。配置手順は [assets/fonts/README.md](../assets/fonts/README.md)。旧「回廊文字グリフ18種」の画像セットは**不要**になった。
- HUD：点灯済みのトラウマ文字（T·S·Ä·C·N·I·G·E·D の点灯順）を黒文字体・琥珀色で表示。未点灯は非表示。
- **R13収束（`VerlustRevealScreen`）**：黒背景。断片→アナグラム整列→`GEDÄCHTNIS`（抑圧 **H** が血色で嵌まる）→赤い `VERLUST` が割り込む。背景画像は不要（テキスト演出）。任意で `bg_void.png`（灰色の虚無）を用意すると D 結末と共用可。

## 8. AI生成のコツ（方針：素材サイト＋AI生成）

- **統一プロンプト核**（例）：`dark vintage 1950s western mansion interior, muted teal-sepia palette, dim cinematic lighting, subtle medical/uncanny undertone, faint signs this is a reconstructed memory, painterly, vertical 9:16`。全背景で共通化し画風固定。
- 4方向は「同一部屋の北/東/南/西」を指定し、**家具配置の連続性**を保つ（同じ部屋に見えるように）。
- オブジェクト/アイテムは透過（`on transparent background`）で生成 or 背景除去。
- 後半部屋は `with faint red digital glitch / corruption creeping in` を追加。
- ⚠️ **商用ライセンス**（広告収益＝商用）を素材側・AI側の双方で確認。img2img の元素材改変が許諾範囲か要確認。
