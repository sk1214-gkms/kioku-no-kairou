# テスト手順書：第1章（1〜10号室）

第1章が完成（ノーマル＋ハード）。実機/エミュレータで通しテストするための手順とチェックリスト。

> ⚠️ このコードは**この開発環境では一度もコンパイル/実行されていません**（Flutter SDK 未導入）。
> 最初の `flutter analyze` で**型エラー等が出る可能性があります**。出力を共有してもらえれば修正します。

## 0. セットアップ（初回のみ）
```bash
# Flutter SDK 導入後、プロジェクト直下で
flutter create .         # android/ios 等の雛形生成（lib/ と pubspec は残る）
#   ※ pubspec.yaml が初期化されたら dependencies と assets(data/) を戻す
flutter pub get
flutter analyze          # まずは静的解析。エラーがあればここで判明
flutter run              # 実機/エミュレータで起動
```
広告は既定オフ（`AdService.enabled=false`）なので、AdMob 設定なしで起動できます。

## 1. スモークテスト（最優先）
- [ ] タイトルが表示される（ノーマル/タイマー/ハードの3ボタン）
- [ ] ノーマルで開始 → STAGE 1 が表示される
- [ ] 1号室をクリアできる（絵画タップ→`1024`）
- [ ] 物語テキスト→「次へ」→ STAGE 2 へ進む

## 2. ノーマル：各部屋の正解
| 部屋 | 操作 | 正解 |
|---|---|---|
| 1 目覚めの暗号 | 絵画を調べる→テンキー | `1024` |
| 2 色の並び替え | 床の本＝手がかり→ボタン | 青→黄→赤→緑 |
| 3 時計の針 | スマホの現在時刻→テンキー（時計03:45は罠） | `0415` |
| 4 鏡の反転 | 鏡/壁→文字入力 ＋ **引き出しで遺留品** | `2108` |
| 5 消えた明かりと選択 | ブレーカーoff→2択(隠す/調べる) | 選択で物語が分岐(confront±1) |
| 6 肖像画のチェス | 肖像＋棚→順タップ | 歩歩歩→騎→城城 |
| 7 薬品棚 | レバーon→影の数字→テンキー | `3514` |
| 8 血痕のABCD | 壁の対応→図形を順タップ | ◯☆□△ |
| 9 焦げたカレンダー | カレンダー＋メモ→テンキー | `1016` |
| 10 暖炉の獣たち | 消灯→暖炉でドアノブ→取付→テンキー | `831` |

### 重点チェック（新機構）
- [ ] **7号室**：レバーoff のまま薬品棚を調べると「今は反応しない」。on にすると数字が見える（requires_state）
- [ ] **10号室**：明かりON のまま暖炉→「今は反応しない」。OFF にするとドアノブ入手（requires_state）
- [ ] **10号室**：アイテム欄でドアノブを選択 → 取付部をタップで装着（item_use）。装着前はテンキーで「反応がない」
- [ ] **4号室**：隠し引き出しで遺留品入手 → 所持品に表示

## 3. ハード：各部屋の難化（モディファイア）
タイトル→ハードで開始。**ヒントボタンは出ない**のが正常。
- [ ] **6 indirection**：台座の指示が「逆順（城→騎→歩）」。順タップは `城城騎歩歩歩`
- [ ] **7 cipher + hidden**：右上に**暗号解読書ボタン**が出る。レバーonで薬品棚が回廊文字 → 解読して `3514`
- [ ] **8 scramble**：図形ボタンの**位置がシャッフル**。位置でなく図形で ◯☆□△ の順
- [ ] **9 decoy**：紛らわしい「別のメモ」がある。惑わされず `1016`
- [ ] **10 decoy**：手前に「古い取っ手」（ダミー）。使っても合わない。本物は暗い暖炉の中。最後は `831`
- [ ] **解読書ボタンは 7号室だけ**出る（他のハード部屋では出ない＝モディファイア連動の確認）

## 4. モード/システム
- [ ] **タイマー**：上部に残り時間。10秒以下で赤。時間切れ→虫食いテキスト→強制的に次へ
- [ ] タイマーでヒントを押すと**時間が一時停止**（広告オフ時はヒントが直接出る）
- [ ] **セーブ**：数部屋進めて一度アプリを閉じ、再起動→タイトルに「つづきから（モード・STAGE x）」
- [ ] つづきからで進行が復元される
- [ ] **エンディング**：全部 full でクリア→最終室で推理（正解可）→通報で **A＋/A**。到達後タイトルへ

## 5. 既知の注意（未検証のため）
