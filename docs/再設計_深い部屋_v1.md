# 再設計ブループリント：全編「深い部屋」化＋高難度 v1

> 方針：**13の“深い部屋”（3章）＋最終室**に集約。各部屋＝4視点×ネスト調査×アイテム合成×多段ロックの脱出部屋。難易度を跳ね上げつつ**理不尽にしない**。物語は核（氷室／蓮見鏡介／甥／館＝記憶）を維持して再編。

## 0. 難易度の原則（調査ベース）
- **難しさ＝層を重ねること**：1つ解くと次が開く／**手がかりを部屋中（4壁）から組み合わせる**／隠し（ブラックライト等）／**解釈層**（数字8→文字H 等）／メタ謎（部屋全体が1つの暗号）。
- **公平の鉄則**：①established ルールを変えない ②**ミスリードはする/騙さない** ③**新メカは“単体で教えてから”組み合わせる** ④適度な達成感を挟む ⑤急な跳ね上げを避け段階的に。
- ハード部屋＝クリア率30%未満を目標。**段階ヒント（3段＋広告）＋（任意）スキップ**で詰み防止。

## 2. メカ導入スケジュール（teach → combine）
新メカは必ず**低難度の部屋で単体で教え**、後の部屋で**組み合わせ**て高難度化する。

| 部屋 | 初出メカ（単体で教える） | 以降での組み合わせ |
|---|---|---|
| R1 | 調べる・拾う・使う | 全部屋の土台 |
| R2 | **アイテム合成** | R9で多段チェーン化 |
| R3 | **隠し（暗転/ブラックライト）** | R8/R10で他要素と結合 |
| R4 | **クロス視点結合＋多段ロック** | 章ボスで集約 |
| R5 | **解釈層（数→文字）** | R11/R13でメタに |
| R6 | **論理/消去法** | R12の証言推理に |
| R7 | **レッドヘリング（ミスリード）** | 以降に散布 |
| R8 | **メタ前哨（altReveal/ツイスト）** | R13の超メタへ |

## 3. 部屋ブループリント（4壁テーマ／謎連鎖／物語 正常・虫食い）

### 第1章「目覚め」
- **R1 白い部屋（目覚め）** ★☆☆☆☆　壁：ベッド/洗面台/扉/窓。
  - 連鎖：洗面台ネスト→鍵→扉の小箱に使う→脱出。**基本操作を教える**。
  - 物語 正常「目覚めると両手が赤い。ここはどこだ」／虫食い「赤い手…私が…？」
- **R2 準備室** ★★☆☆☆　壁：薬棚/作業台/ロッカー/扉。
  - 連鎖：断片2つ→**合成**で招待状完成→氷室の名。差出人コードで扉。
  - 物語：氷室からの招待状。なぜ私が呼ばれた？
- **R3 暗室** ★★☆☆☆　壁：配電盤/現像棚/時計/扉。
  - 連鎖：**ブラックライトON→壁に時刻**。止まった時計03:45は罠、スマホ04:15が真。
  - 物語：時間のズレ。あの時何が起きた？
- **R4 書斎（章ボス／分岐①）** ★★★☆☆　壁：肖像画/チェス棚/金庫/扉。
  - 連鎖：チェスの配置＝金庫番号の一部＋肖像の万年筆の数＝残り（**クロス視点結合**）→金庫→ドアノブ→扉。
  - **【分岐①】サイレン：証拠を隠す(逃避−1)／調べる(直面+1)**。

### 第2章「綻び」
- **R5 診察室** ★★★☆☆　**解釈層**（カルテの数字を文字に：患者ID）。
  - 物語：患者名＝蓮見。私は患者だった？ 担当医だった？（正常/虫食いで分裂）
- **R6 記録室** ★★★☆☆　**論理/消去法**（複数の証言・ログの整合）。
  - 物語：音声ログ＝自分の声「先生、もう限界です」。
- **R7 廊下** ★★★★☆　**レッドヘリング**（白い足跡＝真／血の足跡＝罠／革靴の跡＝甥）。
  - 物語：もう一人いた痕跡。運ばれたのは誰だ。
- **R8 鏡の間（章ボス／分岐②・ツイスト）** ★★★★☆　**メタ前哨＋altReveal**。
  - 連鎖：暗転で肖像が**自分の顔**に→「館＝私の記憶」。鏡の反転＋隠しで最終コード。
  - **【分岐②】否定(逃避−1)／受容(直面+1)**。
- **R9 標本室** ★★★★☆　**合成チェーン**（空シリンジ＋薬液→試作薬／毒薬：解釈分裂）。
  - 物語：治療薬か毒薬か。蓮見の動機が割れる。

### 第3章「真実」
- **R10 監視室** ★★★★☆　**クロス結合＋時間圧（タイマー）**。
  - 物語：防犯映像＝03:45 甥がナイフで書斎へ。犯人は私じゃない。
- **R11 手術室** ★★★★★　**メタ（部屋全体が暗号）＋最小ヒント**。
  - 物語：注射の真相＝記憶を消され犯人に仕立てられた／自分が打った（分裂）。
- **R12 証拠保管室（章ボス／分岐③）** ★★★★★　**全手がかり統合**。
  - 連鎖：甥の遺留品（手袋/筒）を集約し真犯人を確定。
  - **【分岐③】矛盾を暴く(直面+1)／目を背ける(逃避−1)**。
- **R13 最後の扉** ★★★★★　**超メタ**（各章で得た鍵語を結集）→『MEMORY』。
  - 物語：この館＝記憶だと確信し、最後の扉を開ける。

### 終章
- **最終室30 玄関ホール**：**推理3問**（犯人=甥／時刻=03:45／凶器=ナイフ、注射器=ミスリード）→ **3択** → **7エンディング＋精神鑑定書**（既存をそのまま活用）。

## 5. 公平ガードレール
- **新メカは初出部屋で単体提示**（R1〜R8で1つずつ）。R9以降で組み合わせ。
- ミスリード（decoy）は**必ず論理で除外可能**に。騙して誤答に誘導しない。
- **3段ヒント＋広告**（既存）を全部屋に。ハード章は最終段でも“核心の一歩手前”まで。
- 各章の終わりに**達成感（章ボス突破＝物語の山）**。
- 章間で難度の急騰を避ける（章頭は必ず一段やさしく）。
