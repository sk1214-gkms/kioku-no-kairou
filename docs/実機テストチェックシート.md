## 0. 起動（ビルド）
```bash
git pull
python tools/gen_placeholders.py && python tools/gen_item_icons.py   # 仮画像（本番3D未導入の間）
flutter create .            # 初回のみ（android/ios殻を生成）※pubspec初期化されたら assets: を再追記
flutter pub get
flutter analyze             # ← 「No issues found!」が目標
flutter test                # ← 期待 13/13（confab_ending_test に①上書き合算ケース追加済）
flutter run                 # 実機/エミュ
```
- ☐ analyze が **No issues**（赤が出たら全文を記録＝要修正）
- ☐ test が **全緑（13/13）**
- ☐ 起動してタイトルが出る

> 注意：`flutter create .` で pubspec が初期化されたら、`assets:`（data/deep_rooms, data/text_ja.json, data/endings.json, data/cipher.json, assets/images/rooms, assets/images/items）を再追記。

## 3. 今回の修正の確認（重点）
### リプレイ系
- ☐ クリア後、結果画面とタイトルに**「最後の審判からやり直す」**が出る
- ☐ やり直し→3問だけ再回答→**別の結末**が取れる（部屋はやり直さない）
- ☐ やり直し中は**タイマー無し＝Dにならない**、確定文言が**「——また、塗り直すのか。…」**
- ☐ **やり直しでB**を引いた時：ループ自動再走せず**タイトルへ**戻る（別の中断セーブが消えない）
- ☐ 初回クリア後、結末コレクションの**「？？？」タップで条件ティーザー**が出る

### 分岐反応・変種謎（周回で確認）
- ☐ R6で①を「甥」に**書き換えた周**：R7の現場が変わる（**掠れた足跡＋真新しい血の足跡**）／R8小鏡が囁く
- ☐ R4で**逃避**した周：R5に「耳の奥」／**直面**した周：R5に「残響」
- ☐ R8で鏡に**勝利**した周：R10に「砂嵐の顔」
- ☐ R3で①を**書き換えた周**：R10の記録が汚染される（変種＝実装後）／R4で逃避した周：R5の刻みが曖昧化（変種＝実装後）

### 錠・難易度の修正
- ☐ **R10 scrub**：保存ボタンは**常時押せる**。誤った刻で押すと「……違う。この刻では、ない。」（＝正解位置がボタンで漏れない）
- ☐ **R6**：夜勤日誌の時刻は**対峙（消去法）を解くまで滲んで読めない**（解いて初めて0345が判る）
- ☐ **ハード**でR1の札をタップ→**手がかりメモに残らない**（読めない演出と一致）
- ☐ **ハード**のR5→メモが「四隅の刻み 15・7・11・4（順序なし）」／R10→時刻を伏せた記録
- ☐ **ハードR12**：扉プロンプトが「並び順は誰も教えない」＝要素に時刻ラベルが無く、調査で時系列を組む

### エッジケース
- ☐ 途中で**アプリをバックグラウンド**→復帰：タイマー再開・状態維持・クラッシュしない
- ☐ **中断→タイトル→つづきから**：同じ部屋の頭から再開
- ☐ **脳死D**の共有文が「クリア！」でなく**「刻限が、尽きた。／到達」**
- ☐ **S結末の題名が「深淵の白」**（コレクション表示と一致）
