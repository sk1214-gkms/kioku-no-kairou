# 画像の理想像（自然言語アートディレクション）

各画像が「どう見えるのが理想か」を日本語で描写。絵師にも生成AIにも渡せる指示書。
枚数・パス・命名は [アセット完全チェックリスト](アセット完全チェックリスト.md)、英語プロンプトは [アセット仕様書 §8](アセット仕様書.md#8-生成プロンプト集画像ごと)。

## 共通トーン（全室に通す"館のDNA"）
- **舞台**：1950年代ゴシックな洋館だが、その実は無機質な医療施設。優雅な木の内装と、冷たいタイル・器具が同居する不気味さ。
- **色**：くすんだ**ティール＆セピア**を基調に、ごく一部だけ**血の赤**が差す。彩度は低め。
- **光**：薄暗いシネマティック照明。単一光源の陰影が濃い。窓の外はいつも**濃霧**。
- **質感**：半写実の絵画調。埃・染み・経年。**人物は描かない**（主観視点＝プレイヤー自身）。
- **通奏低音**：これは死にゆく脳が作った"美しい偽の記憶"。どこか作り物めいた、記憶が再構成された違和感。章が進む（R10〜）ほど、隅に**赤いデジタルのノイズ／崩壊**が滲む。
- **構図**：縦長9:16。中央〜やや下に「調べる焦点（家具/物）」を据え、周囲は暗く落とす。
- **⚠文字の扱い（多言語対応の恒久規則）**：血文字・メモ・数字など**"読ませる文字"は背景に描き込まない**（滲み・掠れ等の「文字らしき気配」までに留め、実文字はエンジン側でテキスト重畳する）。記号（○△□）はOK。詳細は[改善ロードマップ §4](改善ロードマップ.md)。

## 各部屋（R1〜R13）
> 各方向：**主＝タップの焦点**を画面の見せ場に、（副）はフレーバー/囮として小さく。

### R1 白い部屋（T｜覚醒）
精神科の隔離病室。清潔なのに冷たく、両手には乾いた血の記憶。
- **北・洗面台**：古い白磁の洗面台。下扉がわずかに半開き、配管が湿って光る。くすんだ白タイルに薄い水垢。
- **東・ベッド**：簡素な鉄フレームのベッド。皺のシーツに点々と暗い染み。枕元に薄い影。
- **南・扉**：施錠された無機質な白い扉。小さな覗き窓、鍵穴だけが際立つ。
- **西・窓**：鉄格子の窓。外は乳白色の濃霧で何も見えない。冷たい逆光。

### R2 準備室（S｜復元）
薬品の匂いが満ちる準備室。既視感のある棚の配置。
- **北・薬棚**：ガラス扉の薬棚。整然と並ぶ瓶の奥に、破れた紙片が一枚挟まる。
- **東・作業台**：金属の作業台に散る紙片と、半開きの引き出し。ステンレスの冷たい反射。
- **南・扉**：通行証をかざす読み取り機のある扉。緑の小さなランプ。
- **西・ロッカー**：古いスチールロッカー。『蓮見』の名札が掛かり、その裏に何かが隠れている。

### R3 暗室（Ä｜あの夜）★状態差分
赤い安全灯だけの現像暗室。光を当てれば見たくないものまで浮かぶ。
- **北・現像壁**：暗い壁に吊られた現像途中の印画紙。安全灯の赤に沈む（点灯前）。
- **東・止まった時計＆記憶**：03:45で止まった掛け時計。傍らに、あの夜を再生しようとする飛沫の気配。
- **南・扉**：時計仕掛けの錠がはまった扉。針を合わせる文字盤。
- **西・配電盤**：古い配電盤とブラックライトのスイッチ。傍に「一刻＝二時間」の古い覚書。

### R4 書斎（分岐①｜恩師）★状態差分候補
恩師・氷室の書斎。肖像の目がこちらを追う。
- **北・肖像／金庫**：油彩の氷室の肖像。額の裏に鉄の金庫。（副）若き蓮見と氷室が並ぶ古い写真。
- **東・本棚**：医学書がぎっしり。うち一冊の背に、真新しい指の脂が光る。（副）厚い絨毯。
- **南・扉**：ドアノブの外れた扉。取付部の金具がむき出し。
- **西・暗号パネル／カルテ**：壁に留めたカルテと記号の走り書き。（副）床に散乱した数式の書類。

### R5 診察室（C｜処置）
処置を「受けさせられた」診察室。血は処置の量ではない。
- **北・カルテ／壁の数字**：壁に震える手で刻まれた数字の傷。カルテが乱雑に貼られる。
- **東・診察台**：革張りの診察台に不自然な量の血。拘束具の影。（副）別の壁の古い刻み傷。
- **南・扉**：アルファベット4文字を入れる文字錠の扉。
- **西・薬棚**：施錠された薬棚。管理票に『7-305』。扱い慣れた手つきの記憶。

### R6 記録室（N｜消去法）
事件の記録室。証言を突き合わせると、消去法が一人を指す。
- **北・容疑者表**：壁一面の容疑者表と写真、糸で結ばれた相関。中央に「結論」の空欄。
- **東・証言／門記録**：証言A の紙と、門の施錠記録。日時が並ぶ。
- **南・扉**：数字錠の扉。
- **西・証言／夜勤日誌**：証言B と夜勤日誌。『03:45 異音』で筆跡が途絶える頁。（副）改竄の跡＝新しいインク。

### R7 廊下（I｜足跡）
幾筋もの足跡が交差する暗い廊下。本物は一つだけ。
- **北・往復する足跡**：奥へ往き戻ってくる白い足跡。還り道は歩幅が乱れる。壁に凶器の記憶が滲む。
- **東・血の足跡／血文字**：血の片道の足跡と、壁の血文字『往きて、還る』。
- **南・扉**：数字錠の扉。奥から冷気。
- **西・泥の足跡**：庭師の泥の足跡（囮）。（副）歩幅一定の偽の往復。

### R8 鏡の間（分岐②）★状態差分
明かりを消すと、鏡は"本当の私"を映すという。
- **北・肖像**：穏やかな氷室の肖像。（副）曇った小鏡に「お前は救済者だ」の優しい嘘。
- **東・鏡**：大きな姿見。点灯時はただの鏡（消灯で豹変）。
- **南・扉**：鏡と対峙するまで開かぬ扉。
- **西・照明スイッチ**：古い照明のスイッチ。

### R9 標本室（G｜追跡）★状態差分
ホルマリン標本が並ぶ部屋。奥に隠されたもの。
- **北・標本棚**：硝子瓶の脳や臓器の標本棚。奥に空のシリンジ。
- **東・作業台**：薬液の瓶と、注射針の入った引き出し。
- **南・扉**：注射器を挿す穴のある扉。
- **西・隠し戸棚**：閉じた古い戸棚。中で何かの気配が動く（追跡前）。

### R10 監視室（E｜記録）★状態差分
通電させ、あの夜の記録を再生しろ。時間がない。
- **北・記録端末**：消えた記録端末（通電前は暗い画面）。（副）コーヒーの染みたメモ。
- **東・補助モニター**：暗い補助モニター群。（副）ラベル無しの予備テープの山。
- **南・扉**：記録を掴むまで開かぬ扉。
- **西・主電源**：大きな主電源レバー。（副）埃をかぶった配線の束。

### R11 手術室（D｜手順）
無影灯の手術室。部屋そのものが暗号。
- **北・無影灯／消毒槽**：天井の無影灯（消灯）と、傍らの消毒槽。
- **東・カルテ／麻酔器**：術式カルテと、旧式の麻酔器。
- **南・扉**：処置手順を並べる操作盤の扉。
- **西・器具**：トレイに整列した手術器具（メス・持針器）。止血鉗子も紛れる。

### R12 証拠保管室（分岐③）
すべての断片がここで一つになる。決定的瞬間を、再生するか否か。
- **北・証拠箱**：番号札のついた証拠箱。中に甥の手袋。（副）教授の手帳。
- **東・ファイル／日誌**：事件ファイル、夜勤日誌、検視メモが積まれる。
- **南・扉**：時系列を並べる記録盤の扉。
- **西・現場写真**：03:45を捉えた一枚の現場写真（凶行の瞬間）。

### R13 最後の扉（H収束）
バラバラだった符号が一つの言葉へ収束しようとする。
- **北／東／西・刻印**：壁・床・天井・梁・柱に散った断片の刻印（ME/MO/RY と罠のDE/OR/GE）。
- **南・最後の扉**：巨大な文字錠の最後の扉。奥から光が漏れ、崩壊のノイズが最も濃い。

## 状態差分（ギミックで"物が現れる"理想）
- **R3 `_lit`（BL点灯）**：暗室全体がブラックライトの青白い光に。**北の壁に現像された数字/記号**が蛍光で浮かび上がる。
- **R8 `_dark`（消灯）**：室内が闇に沈み、**東の姿見に返り血を浴びた"私"の像**が浮かぶ（枠に赤い滴り、嗤う気配）。通常時の鏡には映らない。
- **R10 `_on`（通電）**：電源が入り、**北の記録端末と東の補助モニターが青白く点灯**（走査線・再生アイコン）。部屋全体がモニタ光で満ちる。
- **R9 `_open`（筒入手）**：**西の隠し戸棚が開き、中は空**（金属筒を取り去った後）。扉が開いた陰影。

## アイテムアイコン（透過・理想）
くすんだ質感、単体で分かるシルエット、背景透過。
- 小さな鍵（古い真鍮、札に『7-305』）／破れた紙片（左・右）／カルテの担当医欄の切れ端／復元しかけのカルテ／通行証（カルテ様式）／受理印つき通行証（朱の印）／受付印（木柄のスタンプ）／空のシリンジ／注射針／薬液（琥珀の小瓶）／針付きシリンジ／薬液入りシリンジ（琥珀が満ちる）／金属製の筒（冷たい銀色）／ドアノブ（真鍮）。

## 結末キービジュアル（任意・理想の一枚）
- **A+ 完璧な偽物**／**A 作話の綻び**：法廷や無罪の光景だが、隅にほつれ・違和感。
- **B 忘却の揺り籠**：同じ回廊の入口へ戻る円環。優しくも出口のない安堵。
- **C 断罪の重圧**：のしかかる影、判決の気配。
- **S 深淵の白**：安楽死の真相＝白い静寂と、封じた筒。過激描写は暗示に留める。
- **True 白慈の審判**：全てを直視した果ての、静かな覚醒（あるいは崩壊）。
- **D 精神の死**：純粋な暗転・灰色の虚無。ほぼ黒。
