## 2. 部屋をまたぐ仕掛け（謎の“背骨”）
1. **トラウマ文字の収集**：各室クリアで `T·S·Ä·C·N·I·G·E·D`（綴り順でなく**誤認順**）が点灯→R13で `Gedächtnis`、抑圧の **H** が血色で嵌まり `Verlust`（喪失）へ収束。
2. **時刻の30分差**：**03:45（捏造／記録が語る嘘）↔ 04:15（真実／時計の針・自分の足跡が語る）**。R3→R6→R7→R10を貫く。
3. **証拠の二面性**：同じ物証が「甥犯人（嘘）」とも「蓮見犯人（真実）」とも読める→②連結盤で“どう読むか”が道徳の選択。
4. **①記憶の上書き**（R3=真相／R6=犯人／R7=凶器）：**直視**すれば `kept_*` を保持（後の推理で“真実”の論拠になる）。**書き換え**れば嘘が容易になる代わりに真実ルートを失う。
5. **分岐（confront 値）**：R4（怒号を聞く/耳を塞ぐ）・R8（鏡の受容/否定）・R12（記録を暴く/目を背ける）で ±1。全直面/全逃避が結末を分ける。

## 3. 推理（30号室＝②証拠連結盤）

覚醒直前、3つの問いに**手元の証拠を連結して**答える。**“直視した記憶”でしか連結できない**論拠があり、嘘を重ねた者は真実を主張できない。

### 使える証拠（プール）
| 証拠 | 入手条件 | 中身 |
|---|---|---|
| `ev_footage` | 常時 | 03:45の防犯映像（侵入者の影） |
| `ev_testimony` | 常時 | 食い違う三人の証言 |
| `ev_self_culprit` | R6を**直視**(kept_culprit) | 「残るのは私だ」という論理の結論 |
| `ev_paperknife` | R7を**直視**(kept_weapon) | 手に残るペーパーナイフの感触 |
| `ev_that_night` | R3を**直視**(kept_truth) | この手が動いたという記憶 |
| `ev_tube` | R9の**追跡**で金属筒入手(has_culprit_evidence) | 標本室の奥の金属製の筒 |

### 3つの問いと選択肢（各択に必要な証拠＝needs）
| 問い | 嘘(lie) | 真実(truth) | 第3の択 |
|---|---|---|---|
| **犯人は誰だ** | 氷室の甥 〔ev_footage〕 | 私自身（蓮見）〔ev_self_culprit〕 | 館の使用人（囮・証拠不要） |
| **凶器は何だ** | 甥のナイフ 〔ev_footage〕 | ペーパーナイフ 〔ev_paperknife〕 | 銀色のシリンジ 〔ev_tube〕 |
| **何が起きた** | 目撃しただけ 〔ev_footage〕 | この手で刺した 〔ev_that_night〕 | 何も憶えていない（囮・証拠不要） |

- **嘘(lie)を選んだ数 = M**（作話完全度の M。3問すべて嘘＝美しい嘘の完成）。
- **真実(truth)を選ぶ**には対応する記憶を**直視**しておく必要がある（直視していなければその択は連結不可）。
- **自覚ある嘘**：真実を知りながら嘘を主張すると矛盾警告（⚠）。
- **シリンジ**択（凶器）＋ R9の金属筒所持 が **隠し結末S** の鍵。

## 4. 結末判定（作話完全度と7結末＋D）

### 作話完全度
```
I = (T × M) × (1 + E/3)
  T = 生存度（脳死カウントダウンの残り割合 0..100）
  M = 嘘(lie)を選んだ推理数（0..3）
  E = 逃避（s_r4/r8/r12_evade）の回数（0..3）
```
→ 高いほど「美しい嘘」を完璧に組み上げたことになる（＝皮肉な“良い”スコア）。

### 結末の決定木（優先順）
| 優先 | 結末 | 条件 |
|:--:|:--:|---|
| 1 | **D**（精神の死） | 脳死カウントダウンが0＝時間切れ |
| 2 | **S**（隠し・深淵） | 凶器でシリンジを選択 **かつ** R9の金属筒を所持 |
| 3 | **True**（真実） | 全問で真実を選択 **かつ** 全分岐で直面（R4/R8/R12 confront×3） |
| 4 | **A+** / **A** | 嘘 M=3（美しい嘘の完成）。全逃避なら A+、それ以外は A |
| 5 | **B** / **C** | 嘘 M<3（作話に綻び）。全逃避なら B、それ以外は C |

- **B** はループ（やり直しを促す“終わらない”結末）。
- 到達した結末はコレクションに永続記録され、タイトルのギャラリーに並ぶ。

## 5. 難易度バランス所見（机上・改修後）

- **下限（N3）**：R1のみ。導入として最低限の操作教習は残す。それ以外は全室 N6 以上。
- **山場（N10–15H）**：R6（消去法＋改竄看破＋暗証＝最難 H15）／R4（記号置換暗号 H14）／R7（足跡論理＋時刻逆算 H14）／R11（5手メタ H14）。
- **共通の硬化軸**：①答え明示の reveal/ヒントを撤去、②多段導出（暗号・逆算・連鎖合成）、③囮（LIE・偽往復・偽コード・捏造イベント）、④ハードは別動作(`hard_only`)で暗号層・手数・tol を上乗せ。
- **詰み防止**：全室に**段階ヒント＋スキップ**を維持。ノーマルは最終ヒントで答えに到達可能、ハードは最終ヒントを封じる（＝同じ謎でも N/H で体感が大きく変わる）。
- **任意の高難度要素**：R9追跡（S鍵）・①の全直視（Trueルート）・全分岐直面。やり込み層向け。

> 次の検証ポイント（**要実機プレイ**）：平均8/12が体感に合うか、**囮が理不尽（解の根拠が辿れない）になっていないか**、R6の改竄看破・R9hardの「影だけ信じる」7手が運ゲーに見えないか、脳死タイマーが硬化後の所要に対して厳しすぎないか。難しくし過ぎた室は N 側を1–2下げる調整余地を残す。
