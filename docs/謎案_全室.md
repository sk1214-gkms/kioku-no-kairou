# 謎案：全室（ストーリー準拠・改良版）

批評（#1汎用すぎ／#3推理が浅い／#4真実が配給／#5単調）を踏まえ、**物語に準拠して答えが“意味を持つ”謎**へ作り替える案。現行の動く骨格は活かしつつ、(a)数字錠を減らし、(b)答えを物語と接続し、(c)謎タイプを多様化、(d)①②とグリフを背骨に据える。

## 0. 設計原則
1. **答えに意味を持たせる**：暗証は事件の事実（時刻03:45／病棟番号／研究年など）から導く。恣意的な4桁を排する。
2. **謎タイプを散らす**：数字入力に偏らせず「合成／反転／順序／状態／論理／メタ」を配分。
3. **フェアプレイ**：各室に“当事者性”の伏線（Aで実装済）。真実は説明でなく発見へ。
4. **teach→combine**：新メカは低難度で単体提示→後で組み合わせ。
5. **背骨＝メタ**：トラウマ文字(GEDÄCHTNIS)の収集／時刻のズレ／証拠の二面性 が部屋をまたいで一本に繋がる。

## 3. 背骨（部屋をまたぐ仕掛け）
- **トラウマ文字の収集**：各室クリアで T·S·Ä·C·N·I·G·E·D が点灯（誤認順）→R13で `Gedächtnis`→`Verlust` 収束。**全13室を貫く一本の謎**。
- **時刻のズレ**：R3で「止まった時計03:45（犯行）」と「現像04:15（=真の時刻）」の30分差を発見→R10映像の03:45、R12の決定的瞬間03:45 と呼応（跨ぎ）。
- **証拠の二面性**：同じ物証が“甥犯人(嘘)”とも“蓮見犯人(真実)”とも読める→②連結盤で「どう読むか」が道徳の選択。

## 4. 各部屋の謎案

### R1 白い部屋（教習・T）
- 謎：4方向を調べ、洗面台下の鍵で扉。**操作の教習**。
- 物語：両手が赤い／爪の血（伏線）。鍵の札に「**7-305**」（＝第7病棟305号。後のR5白衣と一致＝自分が“ここの人間”の跨ぎ伏線）。
- 改良点：鍵札に意味を持たせ、後室と接続。

### R2 準備室（合成＝カルテ復元・S）
- 謎：紙片2枚を合成→**氷室のカルテが復元**。担当医欄に『蓮見』。差出人/受付コード＝復元文の日付で扉。
- 物語：ロッカーの『蓮見』名札。合成が**物語開示**を兼ねる（ただの通行証でない）。

### R3 暗室（隠し＋時刻dial・Ä／①あの夜）
- 謎：ブラックライトで壁に**真の時刻**が浮かぶ→**掛け時計の針をその時刻に合わせる**(dial)と扉。止まった針03:45は罠、真は04:15（30分差＝跨ぎの起点）。
- ①：飛沫血痕＝あの夜の記憶（直視/書き換え）。

### R4 書斎（クロス結合・多段／分岐①）
- 謎：肖像のチェス駒の数＋パネルの並び→金庫→ドアノブ→扉（多段）。
- 物語：本棚に真新しい指の脂（伏線）。**分岐①**＝金庫の怒号（直面/逃避）。

### R5 診察室（数→文字＝叫び・C）
- 謎：壁の数列を A=1 で文字化→『**HELP**』。これは患者（or 蓮見自身）の刻んだ叫び＝diegetic。
- 物語：処置の量でない血／慣れた手つき（伏線）。白衣の番号7305がR1の鍵札と一致（跨ぎ回収）。

### R6 記録室（論理＋①記憶の上書き・N）
- 謎：3容疑者×3証言の消去法→**残るのは「私」**。直視で自力解錠／書き換えで「甥」に（①）。
- 物語：消去法が自分を指す瞬間が核。②の `ev_self_culprit` はここを直視した時のみ入手。

### R7 廊下（ミスリード＋①凶器・I）
- 謎：白/血/泥の足跡。**論理で本物を選別**（血＝引きずり罠、泥＝庭師、白＝自分）。本物の歩数/向きが値に。
- ①：手の感触＝凶器の記憶（ペーパーナイフ／甥のナイフへ書き換え）。

### R8 鏡の間（対峙＝突きつけ尋問／分岐②）★別動詞#1
- 謎：照明OFF→鏡が**返り血の自分**を映し、「お前は無実だ／甥がやった／見ていただけ」と嘘で諭す。各台詞に対し**各部屋で“直視(kept_*)した記憶を突きつけて論破**（逆転裁判式）。直視していなければ突きつけられない＝①と連動。
- 分岐②内包：3つ全て論破＝鏡を砕き**受容**(s_r8_confront, confront+1)／**目を背ける**＝**否定**(s_r8_evade, confront−1)。数字錠を廃し“別動詞(会話)”へ。

### R9 標本室（合成チェーン＋隠しS鍵・G）
- 謎：空シリンジ＋薬液→満タン→扉。**西の隠し戸棚に金属製の筒**（S鍵＝安楽死の真相）。
- 物語：シリンジ＝“救済”か“凶器”か。②の `ev_tube` 入手点。

### R10 監視室（記憶再生＝スクラブ・E）★別動詞#2
- 謎：主電源ON→**記録端末のタイムライン(03:00〜05:00)をスクラブ再生**し、キーフレーム字幕を見ながら**甥がナイフを構える 03:45 を保存**して解錠(scrub)。脳が捏造した“嘘の物証”(ev_footage)を自ら確定させる行為。
- フェアプレイ伏線：**04:15（R3の“真の時刻”）の乱れに、白衣の袖が一瞬映る**＝捏造の隙間から真実(蓮見)が漏れる。気づける者だけが気づく。時間圧（脳死タイマーが効く章）。

### R11 手術室（メタ＝順序・D）
- 謎：手がかりは説明されない。**トレイの器具を“処置の手順”どおりに並べる**(sequence)＝部屋全体が暗号。
- 物語：自分の手が手順を覚えている＝当事者性。

### R12 証拠保管室（証拠を時系列に並べる＋分岐③）
- 謎：集めた証拠（手袋/筒/写真/映像）を**時系列順に並べる**(sequence)→決定的瞬間が確定。
- 物語：**分岐③**＝03:45の記録を“暴く/目を背ける”。暴くと蓮見の凶行が映る。②へ直結。

### R13 最後の扉（超メタ→収束）
- 謎：三方の刻印 ME/MO/RY→『MEMORY』。確定で**アナグラム収束**：T·S·Ä…が `Gedächtnis`（抑圧Hが嵌まる）→`Verlust`。
- 物語：英語の表層『記憶』→独語の深層『Gedächtnis』→『喪失』。

### 30号室（証拠連結盤＝②）
- 各推理（犯人/凶器/真相）に**手元の証拠を連結して固定**。真実は“直視した記憶”でしか連結不可。**自覚ある嘘**は矛盾警告。シリンジ＋筒→S、全真実＋全直面→True。→7結末。
