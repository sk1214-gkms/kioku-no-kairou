## 各部屋の手順（1室ずつ・R1検証後）
1. §12 のその部屋の各ノードを生成（25枚キュー・seed randomize）→ 良いのをローカルへscp
2. `python tools/crop_916.py <入力> assets/images/rooms/<bg名>.png` で中央9:16化・配置
3. r*.json を viewpoint スイート化（ノード＋不可視ホットスポット＋謎配線）＝R1と同じ要領
4. 実機/Web確認 → 差し替え
