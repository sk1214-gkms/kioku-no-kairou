## 割り当ての手順（各部屋そろったら）
1. その部屋の謎に要る対象（例R2：棚/作業台/ロッカー/扉）に合う絵を選ぶ
2. トーン統一＋9:16化して `assets/images/rooms/` に配置
3. r*.json を viewpoint ノード（またはビュー）に組み、対象にホットスポット＋謎を配線
4. 実機/Web確認 → 差し替え
