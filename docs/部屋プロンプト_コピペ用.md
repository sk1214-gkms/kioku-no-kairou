# 部屋の背景プロンプト（コピペ用・穴埋め不要）

各部屋の **枠内をまるごとコピー → Geminiの画像生成に貼る → 出た1枚を保存 → コマンド1回**。それだけで4方向そろいます。
（画風・4壁レイアウト・各壁の描写を**全部埋め込み済み**。`[◯の焦点]` を自分で書く必要はありません）

## 使い方（1部屋あたり）
1. 下の「R1」の枠を**まるごとコピー**して、Gemini（Nano Banana＝画像生成）に貼って生成。
2. 出てきた**1枚の画像を保存**（例：`sheet_r1.png`）。
3. ターミナルで：
   ```
   python tools/slice_room_sheet.py sheet_r1.png --room r1
   ```
   → `assets/images/rooms/r1_north.png / r1_east.png / r1_south.png / r1_west.png` が自動生成。
4. パネル間に黒い隙間が残るなら `--gutter 8` を足す。

> 💡 気に入らなければ、Geminiに「もっと暗く」「ティール寄りに」「side wallを消して」と追い足して作り直し、同じ手順で。
