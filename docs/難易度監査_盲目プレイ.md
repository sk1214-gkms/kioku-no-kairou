# 難易度監査（盲目プレイ・ユーザー目線）

「プレイヤーが画面で読める文章（各ホットスポットの reveal・錠prompt・対峙の台詞・追跡の気配）だけ」で各室を解けるか、理不尽でないか、体感の難易度はいくつか——を、答え・ヒントを伏せて1周ぶん検証した記録（ノーマル・机上）。

> 前提：本ビルドは**押せる場所が常にラベル枠で表示**される（背景画像の有無に関係なく）。＝画像はパズル難易度に影響せず、難しさはロジック側にある。

## 総評（重要）
- **詰み（解けない／ソフトロック）はゼロ。** 囮もすべて「壊れている／時刻が合わない／罠」と画面上で否定でき、論理で除外可能。フェアプレイは保たれている。
- **ただしノーマルの“素の難易度”は目標(平均8)に届いていない。実測平均 ≈ 5.7。**
  原因＝**いくつかの部屋で reveal が“解き方そのもの”を喋っている**（＋1して読め／暗号表をそのまま提示／プロンプトが答えの時刻を明示 等）。＝硬化の主軸「答え明示の撤去」が、ノーマル本文で徹底できていない。
- **ハード側（hard_only/reveal_hard）は別動作で手数・暗号・囮を上乗せしており、こちらは目標(12〜15)に近い。** 監査はノーマルのみなので、差は主にノーマルの“喋りすぎ”。

## 具体的に“喋っている”箇所（ノーマル本文から答え/手順を抜くべき）
- R4 肖像画：`○＝7／△＝3／□＝0／●＝5` を丸出し → **暗号表を断片化/別解読要**に。
- R5 カルテ：`各数に＋1して読め` → **「震える手が一画手前で止まる」止まりにし、＋1は推理させる**。
- R6 扉/日誌：`夜勤日誌が掲げる“事件時刻”=03:45` を直結 → 日誌は「異音」だけにし、**時刻＝凶行時刻という結び付けを推理**へ。手札ラベルも“アリバイの結論”でなく“証言の生データ”に。
- R7 往復足跡：`唯一、両方向に続く` と名指し → **どれが往復かは複数足跡から判定**させる。
- R10 錠prompt：`日誌『異音 03:45』と一致する刻を保存` → **「記録と一致する刻を探せ」までにして 03:45 を消す**（補助モニターも“二度の乱れ”止まりに）。
- R13 刻印③：`意味は『記憶』` → **削る**（断片ME/MO/RY＋囮DEだけで英単語を当てさせる）。

## 修正方針（ノーマルを真に平均8へ）
- 上記6室の reveal から**「解き方／答え」を撤去し、ヒント側（最終段）にだけ残す**（＝詰み防止は維持しつつ素の難度を上げる）。ハードは現状で目標域。
- これにより実測平均は概ね 7〜8 に上がる見込み。やり過ぎたら個別に reveal を一段戻す。

### 正直な限界（平均8に届かない理由）
置換・＋30分加算・順序・断片結合といった**パズルの“地力”は、答えを隠しても体感7前後が上限**。純粋に平均8へ押し上げるには各室にもう一段の層（例：暗号を二重に／時刻を二段計算／囮を増やし排除に推理を要求）が必要で、**未プレイテストでやり過ぎると“理不尽（根拠が辿れない）”に転びやすい**。安全圏は **ノーマル≈7／ハード12〜15**。さらに上げるなら室を絞って慎重に。

## 画像と探索難度（別軸）
- 現状ホットスポットは常時ラベル表示＝**“絵の中から押せる場所を探す”難しさは無い**。
- もし画像で難度を足したいなら、**実画像（物が描かれた背景）＋ラベル非表示の「探索モード」**が必要（art前提の別機能）。プレイテストは art 無しでもロジック難度は判定可能。
